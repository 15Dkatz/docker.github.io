--- title: "Can an Organization account be converted back to a User account?" id: 2293 draftstate: inactive deleted: false layout: docs permalink: /kb/2293/ originalpath: https://success.docker.com/Support/Can_an_Organization_account_be_converted_back_to_a_User_account%3F source: https://success.docker.com/@api/deki/pages/2293/contents tags: - tag: "article:nontechnical" - tag: "stage:reviewed" --- {% raw %}

Once a User account has been converted to an Organization account, it CAN NOT be converted back to a User account.
