If Docker Trusted Registry is displaying a banner with the message: 

Cannot perform security scans because no vulnerability database was found

It means that the vulnerability database is empty, so it's not possible to scan your images for vulnerabilities. Make sure your node can reach https://dss-cve-updates.docker.com/ , or download an offline version to update your vulnerability database .
