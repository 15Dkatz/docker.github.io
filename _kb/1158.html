Question

How do I switch from the old classic UI to the new UI look in Docker Cloud?

Answer

If you're viewing Docker Cloud with the classic UI there should be a "Preview the New Docker Cloud UI and Repository Security Scanning" link at the top of the page as shown below. Click that link to switch to the new UI.
